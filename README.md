## 准备篇
```
防晒物品：来三亚避免不了去沙滩，防晒工作做不好，随时都可能脱一层皮。
1．太阳伞，出太阳下雨两作用。推荐！
2．太阳帽，即透气又防晒的最好不过了。
3．防晒霜，建议带防晒指数高的。
4．晒后修护露，游泳过后会用到。
5．太阳眼镜，可减低强烈阳光对眼睛的伤害，又有风度。

游泳用品：来三亚不可能不下海吧！
1．泳衣，尽量不要选蓝色，因为海水也是蓝的，蓝上加蓝就没特色了。女生们如果秀一下您的身材，穿比基尼吧！
2．泳镜，更好的保护你的眼睛，一般用不着。
3．防水手表：可以在玩潜水和摩托艇清楚时间，防止被骗！
4．发绳，长头发女生在游泳时用到。
5．浴巾，酒店有配，自己可带!



生活日用品：
1．衣物：带几件易干的衣服。三亚昼夜温差较，白天较热，晚上较凉，最好带件长袖。
2．湿纸巾：在没有水洗手时，派上用场。
3．牙刷，牙膏，毛巾酒店有配，可自带。
4．拖鞋，走沙滩及逛街用到。
5．纸和笔，记录自己的游记。
6．零食，MM途中会用到。
7．背包，带一个带背包，里面装小包包。
8．化妆用品：爱美的女性朋友一定不能忘记。
9．SOD蜜，冬天必要时用得上。
10．弹簧秤，在家买一把过来，三亚几乎买不到的。


药物，防止旅途中不适（必备）。
晕车药，防中暑药、止泻药、止痛药、创可贴、眼药水、消毒棉球、抗过敏药、鼻舒、打算吃海鲜还需带牛黄解毒片、氟哌酸、黄连素。


提示：防晒物品、游泳用品、生活日用品这些如果嫌麻烦，到三亚再买，商场（明珠广场、百佳汇、万豪超市、南国超市、福乐多都有卖）、街边店铺都可买的到。.
```

### 美食
```
海鲜：去三亚必吃的就是海鲜，吃海鲜的地方分为三大部分，分别是第一市场、春园广场、火车头广场。第一市场海鲜最为新鲜，交通便利，缺点：拉客妹太多。春园广场早已不如当年，已经渐渐没落了。火车头广场，海鲜相对新鲜，缺点：离市区较远，没什么好逛的。总结：选择第一市场！
第一市场吃海鲜注意事项：
1.注意街上拉客小妹，千万不要搭理她们，否则一直纠缠。
2.选择正确的海鲜加工很重要，一定要选择网上知名的海鲜加工店，比如四川毛哥海鲜！
3.买海鲜注意别被少称和掺水，不要被掉包，建议让海鲜加工店老板带着去买！
三亚特色小吃推荐：
第一市场除了海鲜，还汇聚了三亚各种特色小吃：
1.抱罗粉：第一市场河边很受欢迎的小正抱罗粉挺火的，但是我感觉抱罗粉味道都差不多。
2.清补凉：清补凉是来三亚必吃的一项小吃，我喜欢的味道。
3.虾饼：也是不容错过的美食之一。
4.文昌鸡：其实我觉得文昌鸡味道一般，推荐大家去常常一家叫海南来吧的果木文昌鸡，他们做法相对传统做法不一样，他们好像是烤的，比较入味。
```
