## 游玩篇
#### 大东海
>  三亚最早开发、最具规模的滨海度假区，离市区最近的海湾，消费适中。高中低挡酒店、客栈应有尽有，综合性价比高；

优点：离市区近 性价比高 阳光海滩第一站；

tips:

```
刚到三亚时的兴奋心情别忘了防晒保护;
```


#### 蜈支洲岛
> 三亚必去地，海水清澈透、沙滩细白柔软，水上项目非常齐全堡礁潜水、珊瑚礁潜水、动感飞艇、电动船、豪华环岛游、拖伞、
动感飞艇、自助午餐、海底漫步、环岛摩托艇等等，可根据预算选择全包产品或者其中几项玩耍；

tips:
```
1. 岛上风浪大记得备件外套或换洗衣服；
2. 潜水的时候会突然潜入水底，耳膜很痛就上来了；
3. 任何项目都得注意安全；
4. 如果预算不多可考虑分界洲岛;
```

#### 亚龙湾
> 三亚四大海湾中人气最高，高档酒店、私家沙滩、沙白水清、好山好水、美景美人，日光浴海浪声，
在这里不用再为去哪个景点而烦恼，定好酒店后安静的躺在酒店里享受度假时光足矣；
>

tips:
```
帅哥、靓女、阳光、沙滩, 备好软妹币享受惬意的海滩度假时光；
```

### 美食

> 去三亚必吃的就是海鲜，吃海鲜的地方分为三大部分，分别是第一市场、春园广场、火车头广场。
第一市场海鲜最为新鲜，交通便利，缺点：拉客妹太多。
春园广场早已不如当年，已经渐渐没落了。
火车头广场，海鲜相对新鲜，缺点：离市区较远，没什么好逛的。

tips:
```
买海鲜加工注意事项：
1.注意街上拉客小妹，千万不要搭理她们，否则一直纠缠。
2.选择正确的海鲜加工很重要，一定要选择网上知名的海鲜加工店，比如四川毛哥海鲜！
3.买海鲜注意别被少称和掺水，不要被掉包，建议让海鲜加工店老板带着去买！

三亚特色小吃推荐：
第一市场除了海鲜，还汇聚了三亚各种特色小吃：
1.抱罗粉：第一市场河边很受欢迎的小正抱罗粉挺火。
2.清补凉：清补凉是来三亚必吃的一项小吃，我喜欢的味道。
3.虾饼：也是不容错过的美食之一。
4.文昌鸡：推荐一家叫海南来吧的果木文昌鸡，他们做法相对传统做法不一样，他们好像是烤的，比较入味。
```
